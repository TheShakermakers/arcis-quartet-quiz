# Arcis Chamber Music Quiz Show

A small offline browser game for a saxophone quartet gag.

Live version for phones:

https://theshakermakers.github.io/arcis-quartet-quiz/

Open `index.html` in a browser, or run a tiny local server:

```bash
cd /Users/claushierluksch/Documents/Computerspiel_V1
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

The quiz uses an English chamber-music question bank converted from Claus'
200-question chamber music millionaire JSON file. The active pool contains only
those 200 questions.

Selected factual sources checked while building:

- Britannica: Chamber music, quartet, Razumovsky Quartets, Dissonance Quartet,
  Emperor Quartet
- Holland Festival: Stockhausen Helikopter-Streichquartett premiere context
- Schott Music: Ligeti String Quartet No. 1, "Metamorphoses nocturnes"
- Bela Bartok Complete Critical Edition: Bartok's six string quartets
- Arcis Saxophon Quartett official site and artist pages for ensemble context
